# 🌀 Lavirint – Smajli VS Đavo

HTML5 canvas maze igra sa 30 nivoa i različitim preprekama.

---

## 🎮 O IGRI

Prolaziš kroz 30 nivoa lavirinta.

Cilj je da stigneš do izlaza, izbegneš prepreke i sakupiš što više novčića.

---

## 👾 ELEMENTI

- 😈 Đavo – neprijatelj
- 👻 Duh – pojavljuje se i nestaje
- 🔥 Vatra – opasne zone
- 🌀 Portali – teleport
- 🚪 Kapije – otvaraju se i zatvaraju
- 💰 Novčići – poeni

---

## 🕹 KONTROLE

- Strelice ⬆⬇⬅➡ (računar)
- D-pad dugmad (mobilni)

## 💾 TEHNOLOGIJE

- HTML
- CSS
- JavaScript (Canvas)
- Web Audio API
